DATA IMMERSION & DATA WRANGLING PROJECT

Description:
This project demonstrates how to load, understand, and clean a dataset using Python and Pandas.

Tools Used:

* Python
* Pandase
* Visual Studio Code
* Excel dataset

Dataset:
students.xlsx (contains Name, Age, Marks)

Steps Performed:

1. Load dataset using pandas
2. Display first 5 rows using head()
3. View dataset structure using info()
4. Display statistics using describe()
5. Check missing values
6. Remove duplicate rows
7. Save cleaned dataset

Output:
cleaned_students.csv (cleaned version of the dataset)

Conclusion:
Data immersion helps understand the dataset and data wrangling cleans the dataset for further analysis.
